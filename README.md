# graduation-project
### intro
这是一个基于vue2.0的todolist项目，后台存储使用的是 leancloud JavaScript SDK，具体使用参见文档，本项目实现的是登录注册以及后台存储的功能。
### usage
* 所有的依赖已写入package.json中，可以用`npm install`或`cnpm install`如下命令快速安装依赖：
```
npm install
```
```
cd step-5
```
```
webpack
```
* 也可以使用webpack-dev-server
```
npm run dev
```
```
npm run build
```
